# i-love-you-animasyonlu-pwa-72
Sevgililer Günü veya özel günlerde kullanıcılara özel animasyonlu mesajlar gönderen bir Progressive Web App. Kullanıcıların sevgililerine animasyonlu kartlar, mesajlar ve hatıralar göndermelerini sağlayan bir platform. Modern UI/UX prensipleriyle tasarlanmış olup, Vite, React, TypeScript, Tailwind CSS, Lucide React ve Framer Motion kullanılarak ...
